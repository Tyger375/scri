# SCRI - Script Maker

Make scripts for your project

## How to use?

How to use scri? Read this guide!

#### Create config

Create config file (.scrirc)

```bash
touch .scrirc
```

#### Add script

Add script in .scrirc

```bash
echo "script_name: command" > .scrirc
```

#### Run script

Run script

```bash
scri script_name
```
